3、exprire 给key值 做一个过期时间 ，写的值是过期的秒数  例子:set phoneCode 3343 ,expire phoneCode 60

5、persist 移除key的过期时间，将 key保持为持久状态。  应用场景： 1)开票

9. randomkey 随机出key  抽奖常用的,点名 随机点名程序，被点到的人 移除去



二、Hash 哈希 类型 适合存储对象
增(单个)：hset 例子：hset food name 'apple'
删除 : hdel food name
批量增 (多个) ：hmset food name 'apple'  color 'red'
批量查：hmget human name age
查： hget  例子：hget food age
查所有key：hkeys food
查所有的key he value :hgetall human
hash 字段的自增:hincrby human age 10
key person value 一堆 k v 组合
hash 数据类型的遍历：hscan (不常用)



三、list ----  是一个双端队列结构  先进后出 排序是按照 插入的顺序排序
比如： 猪八戒 孙悟空  唐曾    朝阳    刘备 张飞 关羽
左边增：lpush key xxxx
右边增：rpush key xxxx
遍历：lrange key 0 -1
左边删除：lpop key (按照顺序一个一个删除 )
右边删除：rpop key(按照顺序一个一个删除 )
插入：linsert key before ‘刘备’ 放 ‘朝阳’
